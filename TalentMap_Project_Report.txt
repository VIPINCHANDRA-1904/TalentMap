                    TALENTMAP - AI SKILL GAP & CAREER PREDICTION SYSTEM
                                  PROJECT REPORT
================================================================================


================================================================================
1. ABSTRACT
================================================================================

TalentMap is an AI-based web application designed to identify the skill gap
between job requirements and candidate resumes. The system analyses uploaded
resumes using Machine Learning and Natural Language Processing techniques to
extract candidate skills, predict suitable career fields and job roles, detect
skill gaps, compute a job match score, recommend top hiring companies with
selection probability, and provide personalized learning recommendations.

Additionally, TalentMap features an Interest-Based Learning module that allows
users to explore alternative career paths. When a user enters a desired job
field and role, the system performs live web searches across multiple sources to
dynamically generate an accurate skill roadmap with curated links to free
courses, YouTube tutorials, and official documentation.

The system is built using Python (Flask), Scikit-learn, NLTK, and SQLite, and
offers a modern, responsive dark-themed web interface with glassmorphism design
elements, micro-animations, and interactive UI components.


================================================================================
2. INTRODUCTION
================================================================================

In today's rapidly evolving job market, candidates often struggle to understand
which skills they need to develop for their desired roles. Similarly, recruiters
face challenges in quickly assessing whether a candidate's qualifications match
job requirements.

TalentMap addresses both of these pain points by providing an automated,
AI-driven platform that:

  (a) Parses resumes to extract candidate information and technical skills.
  (b) Predicts the most suitable career field and job role using a trained
      Machine Learning model (Naive Bayes classifier with TF-IDF vectorization).
  (c) Compares extracted skills against a predefined skill database for the
      predicted role to identify skill gaps.
  (d) Calculates a fair Job Match Score using a weighted scoring formula that
      considers exact matches, partial matches, and base confidence.
  (e) Recommends companies that hire for the predicted role, with detailed role
      descriptions, competitive tier classification, and estimated selection
      probability.
  (f) Suggests learning recommendations for missing skills.
  (g) Provides an Interest-Based Learning feature that uses live DuckDuckGo web
      search to dynamically generate accurate skill roadmaps with curated
      learning resources for any user-specified field and role.

The entire system is packaged as a lightweight Flask web application with a
modern, premium UI, making it accessible to students, job seekers, and career
counselors alike.


================================================================================
3. EXISTING SYSTEMS
================================================================================

Several resume analysis and job matching tools exist in the market:

3.1 Traditional ATS (Applicant Tracking Systems)
    - Systems like Workday, Greenhouse, and Lever are used by companies to
      filter resumes based on keyword matching.
    - Limitations: Purely keyword-based; no skill gap analysis; no career
      prediction; not accessible to individual job seekers.

3.2 Online Resume Scanners (e.g., Jobscan, Skillsyncer)
    - These tools compare resumes against specific job descriptions.
    - Limitations: Require a job description to function; do not predict
      suitable roles; no learning recommendations; most are paid services.

3.3 Career Assessment Platforms (e.g., LinkedIn Skills Assessment)
    - Provide skill quizzes and endorsements.
    - Limitations: Quiz-based, not resume-based; do not analyze career fit
      from existing skills; no personalized learning roadmaps.

3.4 Limitations of Existing Systems
    - No single system combines resume parsing, ML-based role prediction,
      skill gap detection, company recommendation with selection probability,
      and web-search-powered learning roadmaps.
    - Most tools are designed for recruiters, not for job seekers seeking
      self-improvement guidance.
    - Existing systems do not allow users to explore alternative career paths
      with dynamically generated, web-verified skill requirements.


================================================================================
4. PROPOSED SYSTEM
================================================================================

TalentMap is proposed as an all-in-one AI Career Intelligence Platform that
combines the following modules into a single web application:

4.1 Resume Upload & Parsing Module
    - Accepts PDF and DOCX resume formats.
    - Extracts plain text using pdfminer.six and python-docx.
    - Uses regex-based heuristics to extract structured information:
      candidate name, education, and experience.

4.2 Skill Extraction Module
    - Cleans and preprocesses resume text (lowercase, stopword removal).
    - Matches extracted tokens against a comprehensive dictionary of 60+
      technical skills spanning programming languages, web technologies,
      databases, DevOps, cloud, ML/AI, and design.

4.3 ML Role Prediction Module
    - Uses a Multinomial Naive Bayes classifier trained on a curated dataset.
    - Skills are vectorized using TF-IDF (Term Frequency-Inverse Document
      Frequency) before being fed to the classifier.
    - Predicts the most suitable job role (e.g., Data Analyst, Software
      Engineer, Web Developer).

4.4 Career Field Mapping Module
    - Maps predicted roles to broader career fields (e.g., Data Analyst →
      Data Science, Web Developer → Web Development).

4.5 Skill Gap Detection Module
    - Compares candidate skills against required skills for the predicted role.
    - Uses both exact matching and partial/keyword overlap to avoid false gaps.

4.6 Match Score Calculation Module
    - Calculates a fair, ATS-inspired Job Match Score using the formula:
      Score = Base_Confidence + ((Exact_Matches + 0.5 * Partial_Matches) /
              Total_Required) * 70
    - Base confidence rewards candidates for having any skills at all.
    - Score is clamped between 20% and 95%.

4.7 Company Recommendation Module
    - Recommends top hiring companies for the predicted role.
    - Each company includes: role-specific description, competitive tier
      (Elite/High/Moderate/Accessible), estimated selection probability, and
      a direct link to the company's careers page.
    - Interactive accordion UI: click to expand company details.

4.8 Interest-Based Learning Module
    - Allows users to input any desired job field and role.
    - Backend performs live web searches via DuckDuckGo (ddgs library):
      * Queries multiple search phrases to aggregate skill requirements from
        12+ web sources.
      * Extracts relevant skills using a comprehensive keyword bank.
      * Searches again for each skill to find real learning resources.
    - Outputs a step-by-step learning roadmap with clickable links to courses,
      YouTube tutorials, and official documentation.

4.9 Database Storage Module
    - Stores all analysis results in a local SQLite database.
    - Tables: users, resume_analysis, job_roles.

4.10 Web Interface
     - Modern dark-themed UI with glassmorphism effects.
     - Responsive design, drag-and-drop file upload, animated progress
       indicators, and interactive company accordion cards.


================================================================================
5. ADVANTAGES
================================================================================

5.1 All-in-One Platform
    - Combines resume parsing, role prediction, skill gap analysis, company
      recommendations, and learning roadmaps in a single application.

5.2 AI-Powered Accuracy
    - Uses Machine Learning (Naive Bayes + TF-IDF) for role prediction rather
      than simple keyword matching.

5.3 Fair Match Scoring
    - ATS-inspired scoring formula that considers partial matches and provides
      encouraging (not demotivating) scores.

5.4 Live Web Search Integration
    - The Interest-Based Learning module searches the real internet to provide
      up-to-date, verified skill requirements and learning resources.

5.5 Company Intelligence
    - Detailed company profiles with role descriptions, competitive tier
      classification, and estimated selection probability.

5.6 Free & Open Source
    - Built entirely with free, open-source libraries. No API keys or paid
      services required.

5.7 Lightweight & Portable
    - Runs as a local Flask server. No cloud deployment required. Works on
      any machine with Python 3.10+.

5.8 Modern, Premium UI
    - Dark-themed glassmorphism design with micro-animations and interactive
      elements, providing a professional user experience.

5.9 Extensible Architecture
    - Modular code structure makes it easy to add new roles, skills, companies,
      or integrate more advanced NLP models in the future.

9. CONCLUSION
================================================================================

TalentMap successfully demonstrates the application of Machine Learning and
Natural Language Processing in solving real-world career guidance challenges.
The system provides a comprehensive, end-to-end solution that bridges the gap
between a candidate's current skillset and their desired career trajectory.

Key Achievements:
  1. Accurate resume parsing and skill extraction from PDF/DOCX files.
  2. ML-powered job role prediction using Naive Bayes classification.
  3. Fair and encouraging match score calculation inspired by real ATS systems.
  4. Rich company recommendations with role descriptions and selection
     probability estimates.
  5. A novel Interest-Based Learning module that leverages live web search
     to dynamically generate accurate, up-to-date skill roadmaps.
  6. A modern, premium web interface with interactive UI elements.

Future Enhancements:
  - Integration with advanced LLMs (GPT, Gemini) for deeper resume analysis.
  - Support for multiple languages and international job markets.
  - User authentication and persistent profiles.
  - Integration with job portals (LinkedIn, Naukri) for real-time job listings.
  - More sophisticated ML models (XGBoost, BERT-based classifiers).
  - Mobile application development (React Native / Flutter).

TalentMap serves as a valuable tool for students, fresh graduates, and career
changers who want data-driven guidance on their professional development
journey.
